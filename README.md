# LogNow! - App de seguimiento de videojuegos

**LogNow!** es una aplicación web orientada a usuarios que quieran llevar un registro de sus videojuegos, compartir opiniones y descubrir nuevos títulos.

> **Nota:** El proyecto se encuentra en fase de planificación. Las funcionalidades y, sobre todo, el stack tecnológico descritos a continuación son provisionales y podrían evolucionar.


## Descripción

El proyecto nace de la necesidad de organizar el caos de videojuegos pendientes, completados o abandonados que todo *gamer* tiene. La aplicación se conecta con una API externa para obtener datos fiables de los juegos y permite al usuario añadir su capa personal de interacción.

### Funcionalidades principales
*   **Catálogo:** Exploración y búsqueda de juegos.
*   **Biblioteca personal:** Estados de juego (*Jugando, Completado, Pendiente, Abandonado*).
*   **Sistema de Reseñas:** Puntuación y comentarios de la comunidad.
*   **Roles:** Diferenciación entre usuarios invitados, registrados y administradores.

## Tecnologías utilizadas

La aplicación utiliza una arquitectura clásica **MPA (Multi-Page Application)** para garantizar robustez y facilidad de despliegue:

*   **Lenguaje servidor:** PHP 8.
*   **Base de datos:** MySQL.
*   **Frontend:** HTML5, CSS3 y JavaScript para interactividad en el cliente.
*   **Servidor Web:** Nginx.
*   **Control de versiones:** Git y GitHub.
*   **Documentación:** Markdown y GitHub Pages.
